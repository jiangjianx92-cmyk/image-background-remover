# MVP 需求文档：Image Background Remover

> 版本：v1.0  
> 日期：2026-03-25  
> 状态：已完成开发

---

## 1. 产品概述

### 1.1 产品定位

**一句话描述：**  
一款极简的在线图片背景移除工具，帮助用户快速获得透明背景的图片。

**目标用户：**
- 电商卖家（需要白底产品图）
- 设计师（需要素材去背景）
- 普通用户（社交媒体配图）

### 1.2 核心价值主张

| 用户痛点 | 我们的解决方案 |
|---------|---------------|
| Photoshop 太复杂 | 拖拽即用，3步完成 |
| 去背景网站广告太多 | 极简界面，无干扰 |
| 担心隐私（上传图片） | 无存储，内存处理 |

### 1.3 成功指标（v1.0）

- ✅ 功能可用：能成功移除常见图片背景
- ✅ 体验流畅：操作步骤 ≤ 3 步
- ✅ 加载快速：首屏 < 2 秒

## 2. 功能需求

### 2.1 用户流程

```
[上传图片] → [点击按钮] → [查看结果] → [下载]
    1步          1步         自动        1步
```

### 2.2 功能清单

#### 🔧 核心功能

| 功能 | 描述 | 优先级 | 状态 |
|-----|------|--------|------|
| 图片上传 | 支持拖拽上传或点击选择 | P0 | ✅ |
| 图片预览 | 显示原始图片预览 | P0 | ✅ |
| 背景移除 | 调用 Remove.bg API 处理 | P0 | ✅ |
| 结果展示 | 显示去背景后的图片（棋盘格背景） | P0 | ✅ |
| 图片下载 | 一键下载 PNG 格式 | P0 | ✅ |
| 错误提示 | 处理失败时显示友好错误信息 | P0 | ✅ |

#### ⚡ 体验增强

| 功能 | 描述 | 优先级 | 状态 |
|-----|------|--------|------|
| 加载动画 | 处理中显示 spinner | P1 | ✅ |
| 前后对比 | 同时展示原图和处理后图 | P1 | ✅ |
| 重置操作 | 可继续处理下一张图 | P2 | ✅ |

### 2.3 交互细节

**上传区域：**
- 默认：虚线边框 + 上传图标 + 提示文字
- 拖拽悬停：边框变蓝色，背景变浅蓝
- 有图片时：隐藏上传提示，显示图片预览

**处理按钮：**
- 默认：蓝色按钮，显示「✨ Remove Background」
- 处理中：禁用状态，显示 loading
- 完成：隐藏

**结果展示：**
- 棋盘格背景（透明区域可视化）
- 左侧原图，右侧结果，中间箭头
- 下载按钮：绿色，醒目

## 3. 非功能需求

### 3.1 性能

| 指标 | 目标 |
|-----|------|
| 首屏加载 | < 2 秒 |
| 图片处理 | < 5 秒（取决于图片大小和网络） |
| 响应式 | 完美支持手机/平板/桌面 |

### 3.2 兼容性

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
- iOS Safari 13+
- Android Chrome 80+

### 3.3 隐私与安全

| 原则 | 实现 |
|-----|------|
| 不存储图片 | 图片在内存中处理，不落盘 |
| 不留日志 | 不记录用户上传的图片内容 |
| 环境变量 | API Key 存储在环境变量中 |

## 4. 技术方案

### 4.1 架构图

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────┐
│   Browser   │────▶│   Next.js API     │────▶│ Remove.bg  │
│  (用户前端)  │     │   /api/removebg   │     │   (API)    │
└─────────────┘     └──────────────────┘     └─────────────┘
      │                      │
      │                  无存储
      │                  (内存)
      ▼
  返回结果给用户
```

### 4.2 技术栈

| 层级 | 技术选型 | 说明 |
|-----|---------|------|
| 框架 | Next.js 15 | App Router, React 19 |
| 样式 | Tailwind CSS v4 | 快速构建美观的 UI |
| 语言 | TypeScript | 类型安全 |
| 背景移除 | Remove.bg API | 成熟稳定的去背景 API |
| 部署 | Vercel | 边缘计算，全球低延迟 |

## 8. 部署指南

### Vercel 部署

1. Push 代码到 GitHub
2. 在 Vercel 导入项目
3. 添加环境变量 `REMOVE_BG_API_KEY`
4. Deploy

### 本地开发

```bash
npm install
cp .env.local.example .env.local
# 编辑 .env.local 添加 API Key
npm run dev
```
